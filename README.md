# Marnie Theme

母のInstagram投稿を活用した、オリジナルWordPressポートフォリオテーマです🧶  
カスタム投稿・ACF・スライドショー・ハンバーガーメニューなどを実装しました

## 機能概要
- Home：背景スライドショー＋キャッチコピー
- Instagram表示：埋め込み＋Gallery遷移ボタン
- About：画像と自己紹介（レスポンシブ対応）
- Gallery：カスタム投稿＋モーダル画像拡大
- Contact：Googleフォーム
